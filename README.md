# 🚀 Projeto: Manipulação de Dados e Tratamento de Exceções

Este repositório contém uma série de scripts desenvolvidos em Python focados no tratamento robusto de erros e manipulação de arquivos, seguindo as diretrizes de desenvolvimento rápido de aplicações.

## 🛠️ Funcionalidades dos Scripts

1.  **Validação de Strings**: Verifica se a entrada do usuário é um nome textual válido, impedindo campos vazios ou caracteres numéricos indesejados.
2.  **Divisões Seguras**: Implementação de lógica aritmética que previne o encerramento do programa por `ZeroDivisionError` ou entradas não numéricas (`ValueError`).
3.  **Cálculo de Fatorial em Lote**: Processa uma lista de dados e calcula o fatorial apenas de inteiros positivos, tratando individualmente cada erro encontrado na sequência.
4.  **Análise de Arquivos TXT**: Realiza a leitura de arquivos, fornecendo métricas de quantidade de linhas e palavras, com verificação de existência do arquivo no diretório.
5.  **Formatador de Títulos**: Lê nomes completos, aplica formatação de capitalização (Title Case) e gerencia o armazenamento persistente em arquivos de texto.

## 📚 Tecnologias Utilizadas
* Python 3.x
* Google Colab (Ambiente de desenvolvimento)
* Blocos `try / except / else / finally`
* Manipulação de I/O de arquivos (`open`, `read`, `write`)
